1447.最简分数

给你一个整数 n ，请你返回所有 0 到 1 之间（不包括 0 和 1）满足分母小于等于 n 的 最简 分数 。分数可以以 任意 顺序返回。

输出

分析

  • 本题主要难点在于判断是否可以约分

解决方法

  • 判断分子分母的最大公约数是否为1

代码

  • 时间复杂度: 枚举分子分母的复杂度为O(n^2);欧几里得算法:O(log^n);总时间复杂度O(n^2*log^n);
  • 空间复杂度：忽略递归带来的额外空间开销，复杂度为O(1)
